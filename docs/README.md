# 项目文档

本目录包含 WebScan AI Security Platform 的详细技术文档。

## 文档列表

### [架构设计文档 (architecture_design.md)](architecture_design.md)

描述了动态POC验证框架的架构设计，包括：

- **核心组件**
  - 智能匹配引擎 (POCMatcher)
  - 动态POC生成器 (AdaptivePOCGenerator)
  - 验证执行引擎 (DistributedExecutor)
  - 评分系统 (ConfidenceScorer)

- **数据流**
  - 从AWVS扫描到POC验证的完整流程
  - 各组件之间的交互关系

- **技术实现**
  - 匹配策略（CVE匹配、服务版本匹配、模糊语义匹配）
  - POC生成逻辑（模板选择、载荷注入）
  - 验证执行（多语言支持、并发执行、沙箱隔离）
  - 结果评分（响应状态、关键词匹配、时间分析、OOB检测）

## 文档使用指南

### 阅读顺序建议

1. **新手入门**
   - 首先阅读项目根目录的 [README.md](../README.md)
   - 了解项目的基本概念和功能
   - 按照快速开始指南运行项目

2. **架构理解**
   - 阅读 [架构设计文档](architecture_design.md)
   - 理解系统的核心组件和数据流
   - 了解各模块之间的交互关系

3. **开发部署**
   - 阅读 [CI/CD流水线文档](ci_cd_pipeline.md)
   - 了解项目的自动化流程
   - 参考部署指南进行生产环境部署
