## 阅读技巧

### 1. 先看路由定义
每个文件开头都有 `router = APIRouter()`，先看路由路径和装饰器，了解提供了哪些API端点。

### 2. 理解数据流
- 请求 → 参数验证 → 业务逻辑 → 数据库操作 → 返回结果
- 关注 `async def` 异步函数，理解并发处理

### 3. 关注依赖注入
FastAPI 使用依赖注入，注意 `Depends()` 参数，理解服务层调用。

### 4. 跟踪错误处理
注意 `try-except` 块和异常处理，理解错误返回格式。

## 关键概念

| 概念 | 说明 |
|------|------|
| **APIRouter** | 路由分组，每个模块一个router |
| **Pydantic模型** | 数据验证和序列化 |
| **异步处理** | `async/await` 处理并发请求 |
| **依赖注入** | 通过 `Depends()` 注入服务 |
| **状态码** | 标准HTTP状态码（200/400/500等） |
