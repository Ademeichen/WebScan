# 测试脚本

本目录包含用于测试项目各个模块功能的脚本。

## 文件说明

### test_database.py
数据库功能测试脚本，用于验证数据库初始化、健康检查等功能。

**功能：**
- 测试 `init_db()` - 基础数据库初始化
- 测试 `health_check()` - 数据库健康检查
- 测试 `init_database()` - 完整数据库初始化（包括管理员账户）

**使用方法：**
```bash
python tests\test_database.py
```

## 数据库命令行工具

除了测试脚本，您还可以直接使用数据库模块的命令行接口：

### 初始化数据库
```bash
python -m backend.database init
```

### 重置数据库
```bash
python -m backend.database reset
```

## 注意事项

1. 运行测试前，确保已安装所有依赖
2. 测试会创建和修改数据库文件
3. 建议在测试环境中运行，避免影响生产数据
4. 测试失败时，请检查日志输出获取详细信息
